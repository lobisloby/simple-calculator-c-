# 🧮 Calculator (C++ & Qt)

A simple yet elegant desktop calculator application built with **C++** and the **Qt Framework**.  

Perfect for learning GUI development, event-driven programming, and building cross-platform desktop applications.

## ✨ Features

- ➕ **Basic Arithmetic Operations**  
  Addition, Subtraction, Multiplication, and Division

- 🛡️ **Smart Input Validation**  
  Prevents errors like division by zero and invalid operations

- ⌨️ **Dual Input Support**  
  Works with both on-screen buttons and keyboard input

- 🎨 **Clean & Responsive UI**  
  Modern, user-friendly interface built with Qt Widgets

- ⚡ **Lightweight & Fast**  
  Excellent performance with minimal resource usage

- 🌐 **Cross-Platform**  
  Runs smoothly on Windows, Linux, and macOS

## 🚀 Getting Started

### Prerequisites

- Qt Framework (Qt 5 or Qt 6 recommended)
- C++ compiler:
  - **GCC** (Linux)
  - **MSVC** (Windows)
  - **Clang** (macOS)
- Qt Creator (optional but highly recommended)
- CMake (if using CMake build system)

### Installation & Running

**1. Clone the repository**

```bash
git clone https://github.com/yourusername/calculator-qt.git
cd calculator-qt
```

**2. Open the project**

- Recommended: Open the project using **Qt Creator**
- Or build manually using CMake

**3. Build and Run**

Using Qt Creator:
- Open the `.pro` file (if available) or `CMakeLists.txt`
- Click **Build** → **Run**

Or via terminal (CMake):

```bash
mkdir build && cd build
cmake ..
make
./calculator
```
